🎓 KEY LEARNINGS
================

What Worked:
  ✓ Random Forests outperformed Gradient Boosting for this data
  ✓ Ensembles beat individual models consistently
  ✓ Weighted ensembles > naive averaging
  ✓ Feature engineering essential for weak signals
  ✓ IQR-based outlier removal effective
  ✓ 10 good models better than 35 mediocre ones

What Didn't Work:
  ✗ High-degree polynomials (overfitting)
  ✗ Very deep neural networks (underfitting)
  ✗ Stacking with meta-learners
  ✗ More models always better (diminishing returns)
  ✗ XGBoost (dependency issues, not significantly better)

Best Practices:
  • Start simple, add complexity gradually
  • Test diverse algorithm families
  • Use proper train/test splits
  • Weight ensemble members by validation performance
  • Monitor for overfitting/underfitting
  • Ensemble 5-15 strong models, not 35+ weak ones
